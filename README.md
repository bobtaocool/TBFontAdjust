## Installation

TBFontAdjust is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'TBFontAdjust'
```
根据当前的UI图，设置正确的TB_UIScreen即可.

详细使用方法见:[简书](http://www.jianshu.com/p/72c3c55429c3)
